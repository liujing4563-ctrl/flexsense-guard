# FlexSense-Guard 历史项目章程

> 本文件仅用于历史追溯。当前章程以 [`docs/01_project/project_charter.md`](../docs/01_project/project_charter.md) 为准。

## 项目目标

开发一套**不依赖负载侧传感器**的机器人柔性关节可信虚拟力觉与安全控制仿真系统，通过电机侧可测信号实现：

1. 负载侧状态与外部扰动力矩的高保真估计
2. 估计结果的实时可信评分
3. 正常运动、柔性振动、外部接触的可靠辨识
4. 场景自适应的模式切换控制

## 工业场景

协作机器人、服务机器人、外骨骼机器人等对力矩感知精度和人机交互安全性有较高要求的柔性关节应用场景。典型痛点：

- 负载侧传感器增加成本、布线和维护负担
- 关节柔性引入振荡，易与外部接触混淆
- 纯模型估计在参数漂移和异常信号下不可靠
- 缺乏可工程部署的"估计—评分—决策"闭环

## 作品形态

纯算法仿真作品，按机器人核心零部件数字化设计参赛。包含：

- **MATLAB/Simulink MIL**：参数化柔性关节数字样机、增广状态观测器、控制模式验证
- **Python 验证层**：特征提取、规则分类、Schema 校验、批量测试
- **C/C++ SIL**：嵌入式目标代码原型、实时性摸底
- **虚拟传感器看板**：状态可视化、场景配置、一键演示
- **智能测试 Agent**：确定性测试编排、故障注入、报告生成

## 技术闭环

```
轨迹/场景配置
  → 双惯量柔性关节 Plant（电机侧 + 负载侧动力学）
  → 电机侧可测信号（位置、速度、电流）
  → 信号健康检测（丢包、卡死、参数失配）
  → 增广 EKF 状态估计（5 维状态：θm, ωm, θl, ωl, τext）
  → 可信评分（创新残差 + 信号健康 → 置信度）
  → 振动—接触规则分类
  → 模式管理（正常跟踪 / 振动抑制 / 安全减速 / 降级）
  → 控制输出与验证
  → 事件触发低幅校准（低可信度时触发）
```

## 必做项

1. 参数化双惯量柔性关节 Plant（含摩擦、饱和、时延）
2. 仅基于电机侧测量的增广 EKF 观测器
3. EKF 与简单扰动观测器基线对比
4. 可信评分与信号健康检测
5. 事件触发低幅校准与更新门控（接受/保持/回退）
6. 正常/振动/接触三分类辨识
7. 四种控制模式及其切换逻辑
8. MATLAB/Simulink MIL 仿真框架
9. C/C++ SIL 移植与一致性测试
10. Python 批量测试与智能测试 Agent
11. 虚拟传感器看板与一键演示
12. 自动报告生成

## 不做项

参见 `scope_and_non_goals.md`。摘要如下：

- 不依赖负载侧传感器硬件
- 不实现深度强化学习或复杂机器学习（最多轻量三分类）
- 不接入 ROS 2、Gazebo、Isaac Sim 等外部仿真框架
- 不涉及多机器人分布式控制或 Kuramoto 同步
- 不实现差分隐私
- 不做工业安全认证
- 不虚构硬件实测数据或未经验证的性能指标

## 成功标准

| 指标 | 目标 |
|------|------|
| 负载侧位置估计 RMSE | < 0.05 rad（相对于负载侧真值） |
| 外部扰动力矩估计 RMSE | < 0.3 Nm（相对于注入真值） |
| 接触检测延迟 | < 50 ms |
| 正常加速误报率 | < 5%（100 次测试） |
| 振动抑制收敛时间 | < 0.3 s |
| 安全停止时间 | < 0.5 s（从检测到停止） |
| C/SIL 与 MATLAB 一致性 | RMSE 差异 < 5% |
| 可信评分与估计误差相关性 | Spearman ρ > 0.7 |

## 降级路线

| 条件 | 降级动作 |
|------|----------|
| 可信评分 < 阈值（如 0.3）持续超过 1 秒 | 切换到降级模式，仅使用电机侧位置直接映射，限制运行速度 |
| 信号健康检测异常（如编码器完全丢失） | 紧急停止，触发安全减速 |
| 外部冲击检测确认 | 切换到安全减速模式，减速后评估是否需要继续运行 |
| EKF 估计发散（创新残差持续增大） | 回退到简单扰动估计器基线，标记低可信度 |
| 72 小时探针未通过 | 缩小范围至纯电机侧控制，取消负载侧估计和接触辨识 |
