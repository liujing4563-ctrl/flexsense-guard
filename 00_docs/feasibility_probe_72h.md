# 72 小时可行性探针计划

> 本文件仅用于历史追溯。当前规范以 [`docs/03_validation/feasibility_probe_72h.md`](../docs/03_validation/feasibility_probe_72h.md) 为准。

## 目标

在 72 小时内回答三个核心可行性问题：

1. **P1：负载侧状态能否从电机侧信号估计？**（区别于直接使用电机侧位置）
2. **P2：外部扰动（正常运动 vs 柔性振动 vs 外部冲击）是否可分辨？**
3. **P3：可信评分是否能有效识别估计失效场景？**

三个探针逐日推进，每个探针有明确的通过条件和降级策略。

---

## P1（第 1 天）：负载侧状态估计

### 输入
- 双惯量 Plant 默认参数
- 正常加减速轨迹（正弦位置跟踪）
- 无外部扰动的空载场景

### 输出
- 三条曲线在同一张图中：
  - 电机侧位置（θm / N，折算到负载侧）
  - 真实负载侧位置（θl，来自 Plant 真值）
  - EKF 估计负载侧位置（θl_est）

### 通过条件
视觉上，θl_est 明显比 θm/N 更接近 θl。量化指标：

- 负载侧位置估计 RMSE < 0.05 rad
- 或至少比电机侧直接映射的 RMSE 降低 30%

### 降级策略
如果不通过：
- 检查 EKF 参数调谐（Q/R 矩阵）
- 检查可观测性条件（线性化系统是否可观）
- 缩小到仅估计 θl（固定 τext=0）简化问题
- 如果仍不通过：接受电机侧直接映射作为基线，项目调整为无负载侧估计方案

---

## P2（第 2 天）：外部扰动初步辨识

### 输入
- 三个场景（由 `generate_probe_scenarios.m` 生成）
  - 场景 A：正常加减速（无外部接触）
  - 场景 B：柔性振动（参数激励或共振）
  - 场景 C：外部冲击（脉冲型外力矩）

### 输出
- 估计外部扰动力矩（τext_est）时间序列
- 创新残差（innovation_norm）时间序列
- 场景标签

### 通过条件
- 场景 A：τext_est 幅度 < 0.5 Nm，残差平稳
- 场景 B：τext_est 显示振荡模式，残差周期性变化
- 场景 C：τext_est 在冲击时刻有明显跳变（响应时间 < 0.1 s）
- 三个场景的 τext_est 曲线应在视觉上可区分

### 降级策略
如果不通过：
- 检查 EKF 增广状态模型是否正确
- 降低分类难度：先做二分类（正常 vs 异常）
- 延长冲击响应评估时间窗口
- 如果仍不通过：放弃外部力矩估计，仅做负载侧位置估计

### 不允许声称
❌ 已完成可靠碰撞检测
❌ 检测率达到 99%
❌ 零误报

---

## P3（第 3 天）：可信评分初步验证

### 输入
- 正常加减速场景 + 三种故障注入：
  - F1：电流信号卡死（τm 固定为常值）
  - F2：编码器丢包（θm/ωm 随机缺失）
  - F3：Observer 参数严重失配（K_s 偏差 50%）

### 输出
- 估计误差（∥x_true − x_est∥）时间序列
- 信号健康标志（各信号独立 + 综合）
- 可信评分（confidence_score）时间序列

### 通过条件
- 无故障时：confidence_score > 0.7
- 故障注入后：confidence_score 显著下降（低于 0.3 或下降幅度 > 0.4）
- confidence_score 的下降与估计误差增大在时序上相关

### 降级策略
如果不通过：
- 简化评分函数：仅使用创新残差单一指标
- 调整评分阈值以适应仿真特性
- 如果仍然不相关：明确 confidence_score 仅作为实验性指标，不作为模式切换依据

---

## 三天结束后总体评估

| 结果 | 行动 |
|------|------|
| P1 + P2 + P3 全部达到通过条件 | 继续推进完整项目 |
| P1 通过，P2 或 P3 未通过 | 缩小范围，仅做负载侧估计 + 基本评分 |
| P1 未通过 | 项目重新定位为纯电机侧控制仿真，取消虚拟力觉部分 |
| 所有探针均无法在 72h 内达到有意义结果 | 整理经验教训，评估是否继续参赛 |
