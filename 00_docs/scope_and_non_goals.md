# Scope and Non-Goals

## 项目范围

### 包括

1. **柔性关节建模（Plant）**
   - 双惯量弹簧-阻尼模型
   - 库仑 + 黏性摩擦
   - 力矩饱和
   - 测量时延
   - 外部接触/冲击注入

2. **状态估计（Observer）**
   - 增广 EKF（5 维状态）
   - 简单扰动观测器基线
   - 电机侧位置直接映射基线
   - Plant 参数 ≠ Observer 名义参数（明确失配）

3. **可信评分与信号健康**
   - 基于创新残差的置信度
   - 信号健康检测（丢包、卡死、噪声异常）
   - 多源融合评分

4. **事件触发校准**
   - 低可信度触发条件
   - 低幅探针信号注入
   - 更新门控逻辑（接受/保持/回退）

5. **振动-接触分类**
   - 基于规则的正常/振动/接触三分类
   - 频域 + 时域特征提取

6. **模式管理**
   - 正常跟踪（PD + 前馈）
   - 振动抑制（陷波 + 增益调整）
   - 安全减速（预设减速曲线）
   - 降级模式（仅电机侧控制）

7. **仿真与验证**
   - MATLAB/Simulink MIL
   - Python 批量测试（pytest）
   - C/C++ SIL（CMake + ctest）
   - 确定性测试 Agent
   - 故障注入与蒙特卡洛

8. **可视化与应用**
   - 虚拟传感器看板（基于 Python 或网页）
   - 一键演示脚本
   - 自动报告生成

### 明确排除（Non-Goals）

1. **不使用负载侧传感器** — 负载侧位置、速度、力矩仅作为评价真值，不作为观测器输入
2. **不实现 Kuramoto 同步** — 不涉及多关节同步或多机器人协同
3. **不实现差分隐私** — 本项目的仿真数据不涉及隐私保护需求
4. **不实现多机器人分布式控制** — 聚焦单关节柔性关节
5. **不将全部六个关节柔性化** — 仅在单关节上验证方法有效性
6. **不堆叠多种观测器** — 不使用 UKF、ESO、DOB、RBF 等多观测器组合；核心为增广 EKF
7. **不使用深度强化学习** — 不采用 DRL 进行控制策略训练
8. **不接入 ROS 2** — 不依赖 ROS 2 中间件
9. **不接入 Gazebo** — 不使用 Gazebo 物理仿真引擎
10. **不接入 Isaac Sim** — 不依赖 NVIDIA Isaac Sim 平台
11. **不实现多 Agent 系统** — 不涉及 Agent 协商或分布式决策
12. **不做工业安全认证** — 不进行 SIL/PL 等级认证或遵循 IEC 61508 等标准
13. **不虚构硬件数据** — 所有实验结果均来自仿真，不声称经过硬件验证
14. **不虚构碰撞检测精度** — 不自称达到 100% 检测率或零误报
15. **不虚构节能百分比** — 不声称能耗降低的具体数值
16. **不虚构性能指标** — 不在未经验证的情况下发布定量性能声明

## 技术约束

- MATLAB 函数必须有输入/输出说明和单位
- Plant 参数与 Observer 参数必须分开定义且数值不同
- 仿真真值（θl, ωl, τext）禁止传入 Observer
- Python 代码使用类型标注和 pytest
- C 代码使用 C11 标准，CMake 构建，所有变量必须初始化
- 不允许机械地将 MATLAB 代码翻译成不可读的 C 代码
