# Risk Register

## 风险清单

### R1: 负载侧状态不可稳定估计

| 属性 | 内容 |
|------|------|
| 影响 | 核心方案失效，项目需要重新定位 |
| 概率 | 中 |
| 监测信号 | EKF 估计发散（状态协方差矩阵异常增长），或 RMSE 持续 > 0.1 rad |
| 缓解措施 | 可观测性预分析；使用基线方法（电机侧映射）作为兜底；减小模型不确定性假设 |
| 触发降级条件 | Probe 1 未通过且降级措施无效 |

### R2: 外部扰动与摩擦混淆

| 属性 | 内容 |
|------|------|
| 影响 | 接触误报或漏检，降低系统可信度 |
| 概率 | 高 |
| 监测信号 | 正常加速时有持续非零的 τext_est，且与加速度相关 |
| 缓解措施 | 摩擦前馈补偿；区分摩擦特征（速度相关）与外部力矩特征（位置相关） |
| 触发降级条件 | Probe 2 中场景 A 的 τext_est 幅度 > 0.5 Nm |

### R3: 正常加速被误判为碰撞

| 属性 | 内容 |
|------|------|
| 影响 | 频繁触发安全减速，系统无法正常工作 |
| 概率 | 中高 |
| 监测信号 | 正常场景下误报率 > 10% |
| 缓解措施 | 引入加速度阈值；使用双时间窗口确认；调整分类规则灵敏度 |
| 触发降级条件 | 误报率 > 20% 且无法通过参数调整降低 |

### R4: 可信评分与真实误差不相关

| 属性 | 内容 |
|------|------|
| 影响 | 可信评分失去意义，模式切换依据不可靠 |
| 概率 | 中 |
| 监测信号 | 故障注入后评分变化与估计误差变化的 Spearman ρ < 0.3 |
| 缓解措施 | 简化评分函数；引入更多信号健康特征；延长统计窗口 |
| 触发降级条件 | Probe 3 未通过 |

### R5: Plant 与 Observer 同模型验证偏差

| 属性 | 内容 |
|------|------|
| 影响 | MIL 结果过于乐观，C/SIL 移植后性能大幅下降 |
| 概率 | 中高 |
| 监测信号 | MATLAB 结果与 C 结果 RMSE 差异 > 10% |
| 缓解措施 | 明确分离 Plant 参数和 Observer 参数；加入参数失配测试；故障注入验证 |
| 触发降级条件 | MATLAB/C RMSE 差异 > 20% |

### R6: 深度学习数据泄漏

| 属性 | 内容 |
|------|------|
| 影响 | 测试集上性能虚假高估 |
| 概率 | 中（如果启用深度学习） |
| 监测信号 | 训练集与测试集分布重叠 |
| 缓解措施 | 按时间序列顺序分割；不使用未来信息构造特征；先不做深度学习 |
| 触发降级条件 | 本项目本阶段不做深度学习 |

### R7: C/SIL 不一致

| 属性 | 内容 |
|------|------|
| 影响 | SIL 测试与 MATLAB 结果不可比 |
| 概率 | 中 |
| 监测信号 | 相同输入下输出差异 > 5% |
| 缓解措施 | 先定点数/浮点数一致性检查；使用相同的数值积分方法；逐函数单元测试 |
| 触发降级条件 | 差异 > 10% 且无法定位原因 |

### R8: App 开发过早

| 属性 | 内容 |
|------|------|
| 影响 | 在前端 UI 上浪费精力，核心算法未验证时 App 没有实际数据可展示 |
| 概率 | 中高 |
| 监测信号 | App 模块代码量 > 核心算法代码量 |
| 缓解措施 | App 模块仅占位 README，不编写实际代码，直到核心算法通过 Probe 验证 |
| 触发降级条件 | 本阶段禁止 App 开发 |

### R9: 项目范围再次膨胀

| 属性 | 内容 |
|------|------|
| 影响 | 项目无法在截止日期前完成 |
| 概率 | 高 |
| 监测信号 | Issues 中新增的功能请求超出 scope_and_non_goals.md |
| 缓解措施 | 所有新功能请求必须经过 scope 评估；严格区分 P0/P1/P2 优先级 |
| 触发降级条件 | 团队无法就范围收缩达成一致时，项目负责人做出最终决定 |

### R10: 硬件能力被夸大

| 属性 | 内容 |
|------|------|
| 影响 | 虚假的参赛承诺，损害团队信誉 |
| 概率 | 低（本阶段无硬件） |
| 监测信号 | 任何声称"已在硬件上验证"的表述 |
| 缓解措施 | 所有 README 和文档中明确声明"纯仿真作品"；禁止虚构硬件数据 |
| 触发降级条件 | 发现任何虚假声明立即从仓库中移除 |

### R11: 电流信号与真实力矩偏差过大

| 属性 | 内容 |
|------|------|
| 影响 | Observer 模型输入误差大，估计精度下降 |
| 概率 | 中 |
| 监测信号 | τm 估算值与 Plant 真实力矩偏差 > 10% |
| 缓解措施 | 在仿真中加入电流噪声和力矩常数误差；评估不同噪声水平下的估计性能 |
| 触发降级条件 | 在标称噪声水平下 Probe 1 无法通过 |

### R12: Octave 与 MATLAB 不兼容

| 属性 | 内容 |
|------|------|
| 影响 | 团队中部分同学使用 Octave 时代码无法运行 |
| 概率 | 中 |
| 监测信号 | MATLAB 函数在 Octave 下报错 |
| 缓解措施 | 避免使用 MATLAB 特定工具箱函数；使用基础 MATLAB 语法；标注已知不兼容项 |
| 触发降级条件 | 核心功能在 Octave 下不可用且无法替代 |
