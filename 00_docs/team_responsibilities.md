- 能耗—安全—性能权衡分析
- 系统集成、测试报告和答辩准备

`03_confidence_trigger/`, `common/schemas/`, `05_control/mode_manager.m`, `tests/`

## 协作原则

1. 每周至少一次代码评审
2. 模块间通过 JSON Schema 定义的数据接口通信
3. 每个同学的代码变更通过 PR 提交，至少一位其他同学审查
4. 本阶段（仓库初始化 + 72h 探针）由项目负责人同学主导，其他同学熟悉代码库
