# 结果与证据索引

本目录保存可进入版本控制的小型汇总、证据索引、失败记录和阶段决定。原始证据
由产生数据的模块同学负责，项目负责人审查有效性，计算机软件同学维护展示结构。

- `p1/`、`p2/`、`p3/`：阶段证据；
- `integration/`：跨语言和集成验收；
- `reports/`：由真实证据生成的报告产物；
- `generated/`：本地临时生成数据，默认被 Git 忽略。

当前仅建立结构，没有生成新的实验结论。详细规则见
[`docs/03_validation/evidence_management.md`](../docs/03_validation/evidence_management.md)。
