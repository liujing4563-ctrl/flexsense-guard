# Calibration Strategy

## 目的

当可信评分低于阈值时，注入低幅探针信号以激励系统动态，帮助 EKF 重新收敛或校准参数。

## 原则

1. **低幅**: 校准信号幅度不超过名义力矩的 5%，避免对正常运动造成显著干扰
2. **短时**: 持续时间 3~5 个采样步（3~5 ms @ 1 kHz）
3. **稀疏**: 两次校准之间至少有冷却时间（缺省 2 秒）
4. **可检测**: 校准信号有确定性的频率/幅度特征，便于从正常运动中区分
