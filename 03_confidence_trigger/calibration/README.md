# 事件触发校准策略

## 目的

当可信评分低于阈值时，注入低幅探针信号以激励系统动态，帮助 EKF 重新收敛或校准参数。

## 原则

1. **低幅**：校准信号幅度不超过名义力矩的 5%，避免显著干扰正常运动。
2. **短时**：持续 3 到 5 个采样步，在 1 kHz 下为 3 到 5 ms。
3. **稀疏**：两次校准之间至少保留冷却时间，默认 2 秒。
4. **可检测**：校准信号具有确定的频率和幅度特征，便于与正常运动区分。
