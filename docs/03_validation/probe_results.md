# 历史技术探针执行记录

## 记录解释

本文件只保存历史执行证据，不维护当前阶段状态。当前结论见
[`current_status_and_next_steps.md`](../current_status_and_next_steps.md)。旧实验存在力矩
语义、输入一致性、覆盖范围和调参与评价隔离问题，因此不能用于判断修正后路线的
可行性。

## 执行环境

- 运行环境：MATLAB R2024a。
- 场景：`nominal_accel_decel`，采样时间 0.001 s，持续时间 10 s。
- 随机种子：42。
- 估计器输入：仅电机侧位置、速度和力矩；负载侧真值仅在离线指标计算时读取。

## P1：负载侧状态估计

预先定义的过程噪声尺度为 `1`、`1e-2`、`1e-4`、`1e-6` 与 `1e-8`。验收需要
同时满足：负载侧位置 RMSE 小于 0.05 rad，且相较电机侧直接映射至少改善 30%。

| 指标 | 结果 |
|---|---:|
| 电机侧直接映射 RMSE | 0.002584 rad |
| 最佳 EKF RMSE | 3.626873 rad |
| 最佳过程噪声尺度 | `1e-8` |
| 相对改善 | -140248.73% |
| 历史 P1 执行结论 | `FAIL` |

结果只说明旧实现和旧实验配置未通过当时门限。由于证据基础存在确定性缺陷，不能
据此判断修正后的增广 EKF 是否优于电机侧直接映射。
