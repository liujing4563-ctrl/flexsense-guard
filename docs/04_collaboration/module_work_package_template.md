## 当前状态

三类状态分别描述规范、交付物实现和实际验证。准备文档通过检查，不代表主体算法
实现或阶段门通过。

- Specification Status：`DRAFT`、`REVIEWED`、`APPROVED` 或 `FROZEN`；
- Implementation Status：`MISSING`、`PLACEHOLDER`、`PARTIAL`、
  `IMPLEMENTED` 或 `INVALID`；
- Verification Status：`NOT_RUN`、`NOT_VERIFIED`、`PASS` 或 `FAIL`；
- 阻塞项：`<依赖、问题或所需真人决定>`；
- 最后更新：`<YYYY-MM-DD>`。

准备工作不得标记为主体算法 `IMPLEMENTED`，没有有效证据不得标记为 `PASS`。
