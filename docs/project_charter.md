# 项目章程

## 目标

FlexSense-Guard 验证：在单个主导柔性关节的双惯量仿真中，是否能仅用电机侧
位置、速度、电流和力矩信号，估计负载侧状态与外部扰动力矩，并给出可解释的
工程可信评分。

## 阶段门

PR-0 只交付仓库规范、数据契约和 Python 导入检查。72 小时探针通过前，不把
分类、控制、SIL、界面或测试 Agent 作为已完成能力。

## 成功条件

探针应给出 `pass`、`fail` 或 `inconclusive` 的可复现结论，而不是预设性能结论。
若 P1 未通过，项目降级为电机侧控制仿真；若 P2/P3 未通过，缩小为负载侧状态
估计与基础评分。

## 声明边界

项目是仿真研究，不含硬件验证、工业安全认证或未经验证的量化性能声明。公开
仓库不存放专利或比赛敏感材料。
