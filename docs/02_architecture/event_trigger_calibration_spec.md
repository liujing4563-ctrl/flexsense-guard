# 事件触发校准规范

## 1. 规范状态与目的

设计状态：`SPECIFIED`，算法状态：未实现。校准用于在受控、可回滚且证据充分的
条件下评价候选参数，不能用于绕过 P1、在线追逐最终评价真值或掩盖模型错误。

## 2. 允许触发条件

- 系统启动后的受控校准阶段或计划维护窗口；
- 工具、负载或已登记配置发生变化；
- 创新残差持续升高且输入仍有效；
- 可信评分持续下降但尚未进入禁止状态；
- 当前参数在独立监测窗口内持续不满足已冻结的适用性条件。

触发条件只生成候选事件，不代表允许执行或更新参数。

## 3. 禁止触发条件

- 分类状态为 `EXTERNAL_CONTACT`；
- 当前模式为 `SAFE_SLOWDOWN` 或 `LOW_CONFIDENCE_DEGRADED`；
- 编码器无效或时间戳无效；
- 执行器持续饱和；
- Observer 发散、输出非有限或关键输入陈旧；
- 当前没有经过校验的参数备份；
- 上一事件仍在执行、验收、冷却或回滚；
- 候选参数越过物理边界；
- 试图使用最终评价集真值选择参数。

## 4. 冷却时间

每次 `UPDATE`、`HOLD` 或 `ROLLBACK` 后都必须进入冷却阶段。冷却期间可以继续
记录健康状态，但不得启动新候选。具体冷却时间为
`TBD — requires validated Plant and P1 evidence`。

## 5. 最小事件间隔

任意两次校准事件之间必须满足最小间隔，并防止同一原因重复触发。具体间隔为
`TBD — requires validated Plant and P1 evidence`。

## 6. 候选参数

候选范围可以包括负载惯量、刚度、阻尼、摩擦或力矩换算等已经进入冻结模型的名义
参数。每个候选必须声明物理上下界、变化率、所有者和适用版本。新增参数类别必须
先更新模型和接口规范，不得临时塞入校准器。

## 7. 校准前检查

执行前必须确认信号有效、Observer 有限、模式允许、无外部接触、执行器具有余量、
当前参数备份可读且校验通过、调试数据与评价数据隔离，并记录 Git 提交、配置、
随机种子和触发原因。任一检查失败时决定为 `HOLD`。

## 8. 校准执行窗口

激励幅值、持续时间、采样窗口、允许动作范围和停止条件均为
`TBD — requires validated Plant and P1 evidence`。执行窗口必须可中止；出现无效
输入、饱和、接触、发散或安全模式抢占时立即停止并进入 `ROLLBACK` 或 `HOLD`。

## 9. 候选参数验收

候选参数只在预先冻结且与生成窗口隔离的验收窗口中比较。比较必须覆盖有限性、
残差、状态估计指标、物理约束和模式行为；不得只比较单一最好指标，也不得使用
最终评价集选择候选。

## 10. UPDATE

只有候选满足全部强制门禁、证据有效且在冻结比较规则下优于当前可信版本时才可
`UPDATE`。更新必须原子完成，并保存旧版本、候选版本、证据和恢复入口。

## 11. HOLD

证据不足、改善不显著、仍在冷却、前置检查失败或异常无法归类时执行 `HOLD`。
`HOLD` 不得修改当前可信参数。

## 12. ROLLBACK

候选导致失稳、估计退化、非有限、物理越界或模式异常时执行 `ROLLBACK`，恢复
最后可信备份。回滚不是把状态或参数清零，也不能删除失败候选的证据。

## 13. 备份要求

备份必须包含参数值、Schema/算法版本、Git 提交、创建时间、校验值和适用配置。
写入候选前先验证备份可恢复；没有有效备份时禁止校准。

## 14. 失败处理

失败时停止激励、保留失败证据、恢复可信参数，并根据输入有效性进入
`LOW_CONFIDENCE_DEGRADED` 或由有效接触证据进入 `SAFE_SLOWDOWN`。未分类异常
默认 `HOLD`；已应用候选出现异常时默认 `ROLLBACK`。

## 15. 记录和证据

每次事件保存事件 ID、触发与禁止条件、模式、原因码、输入窗口、旧参数、候选参数、
备份校验值、验收窗口、指标、决定、失败原因、提交标识和实际运行状态。未运行的
校准检查必须写 `NOT RUN`。

## 16. 与 Mode Manager 的关系

Mode Manager 对校准具有抢占权。只有允许的稳定模式可开始校准；进入
`SAFE_SLOWDOWN` 或 `LOW_CONFIDENCE_DEGRADED` 时立即中止。Calibration 不得直接
改变运行模式，只能提交带原因的事件结果。

## 17. 与 Confidence 的关系

Confidence 可以提出“需要评估”的证据，但不能单独批准参数更新。硬失效或低可信
降级禁止校准；校准后必须重新经过独立可信与验收窗口，不能立即宣告恢复。

## 18. 尚未冻结的参数

所有激励幅值、持续时间、冷却时间、最小事件间隔、参数步长、验收阈值和窗口均为
`TBD — requires validated Plant and P1 evidence`。本规范不保留 3 至 5 ms 或其他
没有验证证据的固定结论。
