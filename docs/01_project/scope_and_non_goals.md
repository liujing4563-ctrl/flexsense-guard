# 项目范围与非目标

## 研究范围

项目只研究一个主导柔性关节，采用电机侧与负载侧双惯量模型。运行时输入限于
电机侧位置、速度、电流、力矩、力矩指令及其有效性标志。

范围内工作包括：

- 双惯量柔性关节数字样机；
- 摩擦、饱和、时延、负载变化、噪声和故障注入；
- 负载侧位置、速度、扭转量和外部关节扰动力矩估计；
- 正常、柔性振动、外部接触和低可信状态分类；
- 正常跟踪、振动抑制、安全减速和低可信降级模式；
- 工程可信评分、事件触发校准和自动回退；
- MATLAB/Simulink MIL、Python 独立参考、C/C++ SIL；
- 展示界面、自动实验、结果报告和测试 Agent；
- 可复现实验、阶段门、风险控制和跨模块接口。

上述范围按阶段门推进。列入范围不等于当前已完成。

## 数据边界

- Plant 真实参数与 Observer 名义参数必须分离。
- 力矩命令、实际施加力矩和测量力矩必须分离；Observer 不得直接使用未经执行器
  模型处理的原始力矩指令。
- `true_load_position`、`true_load_velocity` 和 `true_external_torque`
  只能进入评价模块。
- 仿真真值不得进入 Observer、Confidence、Classification、Mode Manager 或
  Control。
- `confidence_score` 和 `contact_score` 都是工程评分，不是概率。
- 禁止公共接口字段 `contact_probability`。
- 未声明 Jacobian、等效力臂和坐标变换时，不得把外部关节扰动力矩称为末端力。

## 明确非目标

- 多关节或六关节全部柔性化；
- Kuramoto 振子网络；
- 差分隐私；
- 多机器人协同与分布式控制；
- 同时堆叠 EKF、UKF、ESO、DOB、RBF 等多种观测器；
- 强化学习控制；
- 神经网络直接输出控制命令；
- ROS 2、Gazebo 或 Isaac Sim 集成；
- 真实机器人硬件性能声明；
- 工业安全认证；
- 没有评价模型支撑的末端接触力声明；
- 未经实验验证的碰撞检测率、零误报或实时性宣传；
- 专利、比赛敏感材料和个人数据的公开存储。
